## üéØ Key Features

### 1. **Job Grouping** 
- Automatically merges duplicate jobs (same brand, job name, and date/time)
- **417 duplicates merged** ‚Üí 1,157 unique job entries
- Grouped jobs show with a **red badge** (e.g., "Jobs: 6890, 6892, 6893")
- All models from duplicate jobs are combined and deduplicated

### 2. **Horizontal Table Layout**
5-column spreadsheet-style view:

| Column | Content | Features |
|--------|---------|----------|
| **Job Info** | Job IDs, Brand, Job Name, Date, Duration | Grouped badge indicator |
| **Inquiry Text** | Client request text | Scrollable, language badge |
| **Keywords** | Concept + Model keywords | Color-coded tags |
| **Concept Photos** | Reference images | Horizontal scroll |
| **Selected Models** | Headshots + thumbnails | Horizontal scroll |

### 3. **Integrated Keywords** üè∑Ô∏è
- **Concept Keywords** (blue tags): Extracted from job text
  - Examples: `lookbook`, `lip`, `beauty`, `photography`, `studio`
- **Model Keywords** (purple tags): From style tags + profiles
  - Examples: `commercial`, `casual wear`, `slim`, `brown eyes`, `tall`
- All keywords are **privacy-safe** (no images sent to AI)

## üí° Tips & Tricks

### Quick Filtering
- Search by product type: `lip`, `cosmetic`, `apparel`
- Search by shoot type: `lookbook`, `editorial`, `campaign`
- Search by model style: `commercial`, `casual wear`, `minimal`

### Finding Similar Jobs
1. Note the keywords from a job you like
2. Search for those keywords
3. All similar jobs will appear

### Checking Model Usage
1. Find a model's talent ID
2. Search by that ID
3. See all jobs they've been booked for

### Quality Control
- Search `no keywords` to find jobs missing data
- Use grouped badge to spot duplicate entries
- Check if similar jobs used similar model types
